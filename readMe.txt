----------------------------------------------------
To set up this project you must have node js installed -v 18.13

Download the stable version:
https://nodejs.org/en/

1. Open the terminal 
2. Type 'cd frontend'
3. Then type 'npm install'
4. Close the terminal or cd ... to go back to  the previous folder
5. Type 'cd backend'
6. Then type 'npm install'H
